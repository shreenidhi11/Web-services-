
SETUP: Run the below commands after folder extraction to deploy the application from your environment
----------------------------------------------------------------------------------------------------

1. Open the folder "PA1" in visual studio code or pycharm.

2. Run the command "python -m venv ." to create a virtual environment for this application.

3. Run the command "source bin/activate" to activate the created environment.

4. Run the command "pip install -r requirements.txt" to install all dependencies.
   Note: if you encounter error with pip command use "pip3 install -r requirements.txt"
   instead.

5. Run the command "python server.py" (use python3 if that is installed in your machine)

----------------------------------------------------------------------------------------------------


TESTING: Run the below commands to run the application from your environment
----------------------------------------------------------------------------------------------------

1. After running the server.py file(from pt.5 above), the application will be available on
"http://localhost:8000". Copy paste the link to your browser.

2. Enter a valid genre name and then click on the submit button to get the details. 

3. If no genre is given then default genre's detail will be displayed to the user. In this
case it's python!
